# 02 Home Copy

## 1. 작성 기준

### 강조할 내용

- QA 취업을 가장 중요한 목적으로 두고, 검증·결함 재현·결과 정리 경험을 중심에 둔다.
- 한화비전 SQA/SQE 경험은 실제 수행 업무를 기준으로 구체적으로 표현한다.
- 유앤아이컴퓨넷 장기근속 경험은 성실성, 운영지원, 요청사항 처리, 비용 정리 경험으로 자연스럽게 연결한다.
- 웹사이트 제작 경험은 개발자로 포장하지 않고, 기획·구성·검수·수정 경험 중심으로 표현한다.
- GitHub, HTML, CSS, AI 도구 활용 경험은 "직접 개발 역량"보다 "작업을 이해하고 개선하는 실무 경험"으로 설명한다.

### 피해야 할 표현

- 전문가, 탁월한, 최고, 혁신적인 등 과장된 표현은 사용하지 않는다.
- 실제보다 개발 경험이 많은 것처럼 보이는 표현은 피한다.
- QA 경력과 무관한 사무·운영지원 경험을 앞세우지 않는다.
- 확인되지 않은 성과 수치, 프로젝트 규모, 자동화 수준은 임의로 작성하지 않는다.
- 확인이 필요한 내용은 `확인 필요`로 표시한다.

### 상세 버전

제품이 의도한 대로 동작하는지 확인하고, 결과를 정리하며, 결함을 재현해 관리하는 QA 업무를 중심으로 경력을 쌓아왔습니다. 한화비전 SQA 업무에서는 AI 카메라의 사람, 차량, 얼굴, 번호판 인식 결과를 정탐·오탐·미탐으로 구분해 수치화하고, 펌웨어 성능 평가와 영상 DB 편집 업무를 수행했습니다.

이후 한화비전 SQE 업무에서는 신규 NVR 및 카메라 기능 검증, NVR-Camera 호환성 검증, Regression Test, 결함 재현과 PLM 결함 관리 업무를 경험했습니다. Watson 자동화 환경을 유지하고 단순 스크립트 수정 보조 업무에도 참여하며 검증 환경이 안정적으로 운영되도록 지원했습니다.

유앤아이컴퓨넷에서는 2003년부터 2019년까지 장기간 근무하며 경영지원, ASP 운영관리 지원, 비용 정리, 요청사항 처리 업무를 맡았습니다. 이 경험은 QA 업무에서 필요한 기록 관리, 요청 대응, 반복 업무의 정확성, 협업 과정 이해로 이어지고 있습니다.

최근에는 기존 QA 포트폴리오 사이트, 쵸코네일 홈페이지, 개인 대표 홈페이지를 직접 기획하고 검수하며 웹사이트 제작 흐름을 익히고 있습니다. GitHub, HTML, CSS, AI 도구를 활용해 화면 구성과 문구를 점검하고 수정하는 방식으로 작업했으며, 개발자로 과장하기보다 기획과 검수, 개선 과정에 참여한 경험으로 정리합니다.
